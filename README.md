# Smith and Shin CPAs, LLC

Live at **[smithandshin.com](https://smithandshin.com)**

Business website for Smith and Shin CPAs, LLC — my dad's CPA firm in Taylors, SC. I transferred the domain from the previous registrar to Cloudflare, set up DNS, and built the site from scratch.

## Pages

- **Home** — Firm overview, team bios, and contact info
- **Services** — Tax preparation, tax planning, and consulting details
- **Tax Tips** — Seasonal tax guidance and resources

## Tech stack

- **Frontend:** Static HTML + Tailwind CSS (via CDN)
- **Hosting:** Cloudflare Pages (auto-deploys from this repo on push to `main`)
- **Domain registrar:** Cloudflare
